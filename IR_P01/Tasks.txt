1. Way to run:

   1.1 Download any webpage or create any webpage of your own and keep into any folder. Ex: Documents
   1.2 Build the artifacts.
       $ mvn clean install
   1.3 Run command.
       $ java -jar IR P01.jar /home/anis/Documents /home/anis/index VS "awesome title"
